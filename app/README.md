# App

Webpack build using HTML, SCSS, and JS (ES6 modules).

## Scripts

| Command | Description |
|---|---|
| `npm start` | Start dev server at `http://localhost:6969` with hot module replacement |
| `npm run build` | Production build — outputs to `dist/` with extracted CSS and hashed filenames |
| `npm run dev` | Development build — outputs to `dist/` without optimizations |
